# Creative generator

## Steps

1. `generate_ad_copy` — headlines / primary text / CTA variants
2. `upload_creative` — Meta/TikTok image URL upload (dry_run then confirm)
3. `list_creatives` — inventory (Google assets / Meta images / TikTok / LinkedIn)
4. Optional: `suggest_creative_rotation` then `pause_ad` for Meta losers

## Out of scope

Imagen / Veo / Runway multi-model generation is not available — do not invent those tools.
